# Forex & Crypto Market Analysis Signal Bot

A lightweight Python-based market analysis and signal generation system.

## Scope (MVP)
- Forex & crypto market data
- Technical indicators (EMA, RSI)
- Rule-based strategy logic
- Structured BUY / SELL / HOLD signals
- Chart visualization for signal validation

## Explicitly NOT included
- Live trading
- Order execution
- Broker integrations
- Machine learning (for now)

## Design goals
- Low cost / free data sources
- Simple, readable architecture
- Easy to extend without refactoring
